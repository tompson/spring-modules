For adding your module docs to the official, complete, Spring Modules documentation, please do the following:

1) Add yourself to the authors, modifying the reference/src/authors.xml file.

2) Add your docbook files to the index, modifying the reference/src/index.xml file.

3) Copy your project doc images into the reference/src/project-images directory.

If you have any problem, feel free to ask a question to the Spring Modules dev list: dev(at)springmodules.dev.java.net
