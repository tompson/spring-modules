  • Project Name: Spring Modules
  • Latest Release: 0.1
  • Latest Release Status: Alpha
  • Project Type: Library
Spring Modules is a collection of tools, add-ons and modules to extend the Spring Framework. The core goal of Spring Modules is to facilitate integration between Spring and other projects without cluttering or expanding the Spring core.

Features
The modules included in the 0.1 release are:
  • OSWorkflow configuration support
  • Support for JSR-94 Rules Engines such as Drools and Jess
  • HiveMind integration
  • Commons Validator integration

Build Instructions
The Spring Modules distribution comes complete with binaries and source code. To build the source code yourself simply run ant build from the command line. You can run the test suite on your local machine using ant tests.

Usage Instructions
See the PDF reference manual found in docs/reference/pdf for more information. If you have any questions please post on our mailing lists.

Participation
The best way to participate at this point is to let us know what features you would like to see in Spring Modules. If you have a module of your own then let us know on the mailing list or post a feature request on the mailing list or on JIRA.
